# Reader
## Description
This Task reads two files containing lists of integers and prints pairs of integers from the two files line by line. It stops when the shorter file is exhausted.

### Example
`File A`:
```
1
2
3
4
5
```

`File B`:
```
6
7
8
```

`Output`:
```
1 6
2 7
3 8
```

## Inputs
- `a` - first input file
- `b` - second input file
